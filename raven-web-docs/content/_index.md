**Raven** is a CLI for job discovery, tailored application drafting, and email outreach.

**Discover → Draft → Send**

```bash
./raven setup
./raven discover
./raven draft --max 25
./raven send --input drafts/outreach-2026-07-04.csv --dry-run
```
